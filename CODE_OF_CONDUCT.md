# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project
a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept responsibility for impact.
- Keep discussions focused on the project and its users.
- Give and receive technical feedback with professionalism.
- Make space for contributors with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Sexualized language or imagery.
- Personal attacks or sustained disruption.
- Publishing private information without explicit permission.
- Any conduct that would be inappropriate in a professional setting.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or ban
participants whose behavior violates this code of conduct.

If you experience or witness unacceptable behavior, report it privately to the
maintainers. Until a dedicated security/contact address is published, use a
private maintainer contact or GitHub's abuse reporting tools rather than opening
a public issue with sensitive details.

## Scope

This code applies in project repositories, issue trackers, pull requests,
community spaces, and any public context where someone represents the project.
